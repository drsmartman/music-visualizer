Installation
- Install openFrameworks v0.9.4 or greater
- Build and Run in XCode 8.1 or greater

Add Music
- To add music, in src/ofApp.cpp look for 'beat.load()' and add the local path of your MP3. Also, add the MP3 file to 'bin/data'
